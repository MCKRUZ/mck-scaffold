### CSDD security overlay — methodology notes

This project applies the **CSDD security overlay** on top of MUSUBI (L2 Spec-Anchored). Security is constraint-enforced, not advisory.

**The rule:** for any code change, the SEC-NNN clauses in `docs/constitution.md` are merge-blocking. The `@constitution-enforcer` skill rejects implementations that violate them, with line-level diagnostics.

**Key SEC clauses to know** (full list in `docs/constitution.md`, "Security Clauses" section):

| ID | CWE | Topic | Enforcement |
|---|---|---|---|
| SEC-001 | CWE-89 | SQL injection | Parameterized queries / ORM only; reject string concat |
| SEC-002 | CWE-79 | XSS | Output encoding; no `dangerouslySetInnerHTML` without justification |
| SEC-003 | CWE-798 | Hardcoded credentials | Secrets in vault/env only; reject in source |
| SEC-004 | CWE-502 | Insecure deserialization | No `pickle.loads` on user input; safe-list deserializers |
| SEC-005 | CWE-310 | Cryptographic failure | Use platform crypto libs; reject custom crypto |
| SEC-006 | CWE-22 | Path traversal | Path canonicalization + allowlist before file access |
| SEC-007 | CWE-352 | CSRF | Token validation on state-changing endpoints |

**Compliance Traceability Matrix:** each SEC clause maps to specific lines of code that satisfy it. Auto-generated by `@security-auditor`, lives at `docs/compliance-matrix.md`. Auditors love this artifact. Build the muscle to produce it.

**Workflow addition (on top of MUSUBI):**

1. After `@requirements-analyst` writes REQs, run `@security-auditor` to surface relevant SEC clauses for this feature.
2. Implement against both REQs and SEC clauses.
3. `@constitution-enforcer` runs on commit; static analysis (semgrep / equivalent) runs as the pre-commit hook configured in `.claude/settings.json`.
4. Failed enforcement = no merge. No exceptions without a documented variance ADR.

**Re-evaluate SEC clauses:**

The CWE / MITRE Top 25 / OWASP Top 10 shift annually. Set a calendar reminder for the date in `docs/decisions/METHODOLOGY.md` to revisit which clauses still apply and which need adding.
