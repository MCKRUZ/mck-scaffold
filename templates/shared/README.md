# templates/shared/

Base files written into every scaffolded project, regardless of methodology preset. Methodology presets and stack adapters compose *on top* of these files; they never replace them entirely.

## Files

| File | Purpose |
|---|---|
| `CLAUDE.md.tmpl` | Lean CLAUDE.md base (~25 lines). Plan Mode reminder, methodology/stack pointers, AGENTS.md link, decision-doc links. |
| `AGENTS.md.tmpl` | Cross-tool config for Codex/Cursor/Gemini/Copilot. Points at CLAUDE.md as authoritative. |
| `settings.json.tmpl` | Minimum valid `.claude/settings.json` — empty hook arrays, `attribution.commit: ""`, empty permissions. Presets and stacks append. |
| `docs/decisions/INTERVIEW.md.tmpl` | Permanent record of Q1–Q8 answers. Source of truth for re-run detection. |
| `docs/decisions/METHODOLOGY.md.tmpl` | Why-this-setup rationale, alternatives considered, scaffolded files list, caveats, re-eval cadence. |

## Template variable schema

All templates use `{{variable_name}}` placeholders. The scaffolder (init/SKILL.md, Phase 3+) is responsible for substituting them.

## Composition order

When the scaffolder writes a target project's `CLAUDE.md`:

1. Start with `templates/shared/CLAUDE.md.tmpl`
2. Substitute project context variables
3. Replace `{{methodology_specific_content}}` with `templates/methodologies/<preset>/CLAUDE.append.md` (substituted)
4. Replace `{{stack_specific_content}}` with concatenated `templates/stack/<each-stack>/CLAUDE.append.md` (substituted)
5. Write the resulting markdown to `<target>/CLAUDE.md`

For `settings.json`:

1. Start with `templates/shared/settings.json.tmpl` (empty hook arrays)
2. Merge `templates/methodologies/<preset>/settings.merge.json` (append to hook arrays, merge permissions)
3. Merge each `templates/stack/<each-stack>/settings.merge.json`
4. Apply MCP overlay if Q6 = yes
5. Write the merged JSON to `<target>/.claude/settings.json`

Order rule: shared base → methodology → overlays → stack. Later layers override earlier where keys collide; arrays are appended; objects are deep-merged.
